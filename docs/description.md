# Сонгосон програм

Би агуулахын удирдлагын системийг сонгож дууриалгасан.

## Хийсэн дэлгэцүүд

- Нүүр дэлгэц: агуулахын товч мэдээлэл, сүүлийн хөдөлгөөн, хурдан үйлдлүүдийг харуулна.
- Барааны дэлгэц: бараа хайх хэсэг болон одоогийн үлдэгдлийг харуулна.
- Орлогын падаан: агуулахад бараа хүлээн авах маягтыг харуулна.
- Тайлан: тайлангийн төрлүүд болон жишээ хүснэгтийг харуулна.

## Ажиллах байдал

Энэ нь GUI загвар тул мэдээлэл хадгалахгүй. Товч дарах болон дэлгэц солих үед сонгосон үйлдлийг консоль дээр хэвлэнэ.
