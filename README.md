# Лаб06-2: Swing GUI

Энэ ажил нь агуулахын системийн хэд хэдэн холбоотой дэлгэцийг Java Swing ашиглан дууриалгаж хийсэн GUI загвар юм.

Дэлгэцүүд нь зөвхөн харагдах хэсгийн загвар. Өгөгдлийн сан болон бодит хадгалалт хийгдэхгүй. Товч дарахад сонгосон үйлдлийг консоль дээр хэвлэнэ.

## Хийсэн дэлгэцүүд

- Нүүр дэлгэц
- Барааны үлдэгдлийн дэлгэц
- Орлогын падааны дэлгэц
- Тайлангийн дэлгэц

Дэлгэцийн зургууд `docs/screenshots` хавтаст байгаа.

## Ажиллуулах

```bash
javac src/*.java
java -cp src Main
```

## Ашигласан Swing компонентууд

- `JFrame` - үндсэн цонх
- `JPanel` - хэсгүүдийг байрлуулах
- `JButton` - товч
- `JLabel` - гарчиг, текст
- `JTextField` - утга оруулах талбар
- `JComboBox` - сонголтын жагсаалт
- `JList` - тайлангийн жагсаалт
- `JTable` - хүснэгт
- `JScrollPane` - хүснэгт, жагсаалтыг гүйлгэх хэсэг
